Surgimento da Internet

(Voltar)

Como Surgio a Internet

O conceito da internet teve um fundo militar e, por isso mesmo, os países que primeiro desenvolveram esse conceito foram EUA e URSS. Nos Estados Unidos, no começo dos anos 1960, já havia sido criado um sistema para o Pentágono que possibilitava a descentralização dos documentos. O objetivo era proteger esses documentos de ataques e perdas.

Internet é com a ARPANET (Rede de Agencias de Projetos de Pesquisa Avançada ou Rede da Agência de Projetos de Pesquisa Avançada dos Estados Unidos). No ano de seu nascimento, em 1969, quando nasceu a ARPANET, surgiu a primeira conexão entre os computadores da Stanford e da UCLA. Dois anos depois, o primeiro e-mail foi enviado por Ray Tomlinson. Naquele ano, também aparece o primeiro vírus Creeper. No início dos anos 70, nasceu a palavra Internet, que era aplicada ao sistema de redes interconectadas por meio dos protocolos TCP e IP (Transmission Control Protocol / Internet Protocol) nos quais eles baseavam serviços de Internet e e-mails. Isso tudo até 1990, quando Tim Berners-Lee, conhecido como o pai da web, cria a linguagem HTML. A equipe do cientista criou o primeiro cliente web “WorldWideWeb” (www).

Os Primeiros computadores

1ª Geração - Temos o computador Harvard Mark I (1944), Colossus (1946), ENIAC (1946), todos eram grandes e o manuseio era com válvulas.

2ª Geração - (1959 - 1965) Teve uma importante evolução para a época, a mudança de Válvulas para os Transistores, com isso diminuindo o tamanho dos computadores.

3ª Geração - (1965 - 1970) A criação do Circuito Integrado, diminuindo ainda mais o tamanho dos computadores.

4ª Geração - Que começou em 1971 e continua evoluindo até hoje, marcada pela criação do mouse,pela Apple, que atualmente usamos e computadores com multiprocessadores e Smartphones.

O que são Clients

(Voltar)

Conceito

É o cliente quem sempre inicia a comunicação com o servidor. O cliente mais conhecido é o navegador. Quando acessamos um site pelo navegador, uma requisição é disparada pela internet para o computador no qual esse site está hospedado.

Navegadores

Os navegadores que também são chamados de Browsers , são programas criados por empresas, utilizados para abrir/executar arquivos.Seguem padrões W3C,porém, sempre tem uma diferença ou outra de interpretação, iniciou com o MOSAIC, passou para o Netscape, e hoje temos uma variedade de navegadores disponíveis.

Aplicação Web

São soluções criadas que possuem a internet como meio de comunicação entre Client x Server não sendo necessário a sua instalação.

Sobre Servidores

É um computador com uma grande capacidade de processamento que conecta outros computadores através de uma rede, sendo local ou web. O servidor pode ser pequeno ou até mesmo grande, ocupando uma sala inteira. Existe diversos tipos de servidores, pode ser de arquivos, segurança(Firewall), streaming, email,web.

Hardware e Software em Servidores

Software: Sistemas operacionais, Monitoramento e Servidores Web.

Hardware: Armazenamento(disco rígido, SSDs),Memória,Processadores.

O cliente faz a requisição HTTP (protocolo de transferência de hipertexto), que chega ao servidor, e lá o software do servidor devolve em formato HTML, CSS, Javascript, Imagens, Texto,etc.

Linguagem de Programação

O que é

Linguagem de Programação é uma linguagem escrita e formal que especifica um conjunto de instruções e regras usadas para gerar programas (software).

5 linguagens mais usadas, segundo RedMonk:

1. Javascript

2. Python

3. Java

4. PHP

5. C#

Serve-Side

Linguagens executadas no servidor, como PHP, Ruby, C# e Java, e Javascrip.

Client-Side

São linguagens executadas no lado Client, como Javascript, HTML e CSS

Curiosidade - Javascrip é executado tanto no lado Client quando no lado Server,o Javascript possui uma ramificação que se baseia no interpretador v8 do Google, que pode ser executada no lado servidor, chamada Node.js

HTML na Prática

(Voltar)

Estrutura básica

  • O documento deve iniciar com a tag "html", seguida de "head" com o "title" (título da página) e "body", onde ficará o conteúdo da página.
  • Títulos (h1), subtítulos (h2 em diante) e parágrafos (p).
  • Negrito (strong ou b), itálico (i), sublinhado (u), marcado (mark), elevar algum caractere* (sup, e o caractere).
  • Listas ordenadas com "ol" e não ordenadas com "ul". Cada item da lista recebe a tag "li".
  • A tag "a href="">" serve para colocar links externos (com endereços Web) ou internos (através do id) na página. Os atributo "title" serve para definir um texto a ser exibido ao passar com o mouse no link, e o atributo "target" serve para definir se o link externo será aberto na mesma aba, ou em outra (com o valor blank).
